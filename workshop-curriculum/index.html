Pop Up Workshops

Pop Up Workshops are 2-3 hour standalone workshops where participants will use a starter kit to build a working project! Attending will give you a feel for if coding is for you.

Front-End Day

The Front-End is the part of the application that users see, touch, and interact with. This is the code that produces the experience in front of the user. This day-long workshop will introduce you to real-world Front-End applications and the foundations of the the technologies used to build them.

Get Started

Back-End Day

The Back-End typically handles work like storing information in databases, manipulating that data, signing in users, etc.; it's what happens behind the scenes. This day-long workshop will introduce you to real-world Back-End applications and the foundations of the Ruby programming language.
